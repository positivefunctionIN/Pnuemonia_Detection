# Chest X-Ray Pneumonia Detection with Grad-CAM

![Streamlit App](https://img.shields.io/badge/Streamlit-FF4B4B?style=for-the-badge&logo=Streamlit&logoColor=white)
![TensorFlow](https://img.shields.io/badge/TensorFlow-FF6F00?style=for-the-badge&logo=tensorflow&logoColor=white)

A deep learning application for detecting pneumonia from chest X-ray images with Grad-CAM visualization for model interpretability.

## 🚀 Features

- **Pneumonia Detection**: Custom CNN model trained on chest X-rays
- **Grad-CAM Visualization**: Heatmap overlays showing model's attention areas
- **Interactive Interface**: User-friendly Streamlit web application
- **Real-time Predictions**: Instant analysis with confidence scores
